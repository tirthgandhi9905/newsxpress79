# GUI Testing – Selenium IDE

## 1. Introduction
This folder contains the Graphical User Interface (GUI) test cases created using **Selenium IDE**.  
Selenium IDE is a browser-based automation tool used to record and replay actions on a web application.

---

## 2. Purpose
The aim of these tests is to validate:
- User interface functionality  
- Navigation flow  
- Form handling  
- Buttons, links, and UI components  
- User interaction behaviour  
- Page redirections  

---

## 3. Tools & Environment
- **Tool:** Selenium IDE  
- **Browser:** Chrome / Firefox  
- **Project Type:** Web Application  
- **Test Format:** `.side` files  
- **Test Focus:** Login, Signup, Navigation, CRUD flows and other UI features

---

## 4. How to Run the Tests
1. Install the **Selenium IDE** browser extension.  
2. Open Selenium IDE from your browser.  
3. Import the `.side` test file from this folder.  
4. Click **Run All** to execute test cases.  
5. Check the execution logs, results, and any screenshots (if enabled).

---

## 5. Output
- Pass/Fail result summary  
- Execution logs

- Assertion checks  
- Test execution timeline
